Centos7 Gpu driver

Sep 12, 2019 #centos7 #nvidia gpu驱动 #cuda

在vsphere创建的虚拟机上安装nvidia-gpu驱动时和cuda,输入nvidia-smi时出现错误

Unable to determine the device handle for GPU 0000:0B:00.0: Unknown Error

查询后发现是需要设置一个虚拟机的参数然后重启虚拟机

hypervisor.cpuid.v0 = FALSE

在vsphere上找到虚拟机先关机,关机完成后点击编辑设置-虚拟机选项-高级–配置参数-编辑配置

配置参数hypervisor.cpuid.v0 = FALSE,然后启动虚拟机.

输入nvidia-smi,万事大吉啦.
